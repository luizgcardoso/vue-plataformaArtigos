# Vue Plataforma de Artigos

**Projeto Fullstack em Vue.js - Plataforma de Gerenciamento de Publicação/Acesso a Artigos Online**

Aplicação web para criação, publicação, gerenciamento e leitura de artigos, desenvolvida com Vue.js no frontend e Node.js/Express no backend

## Tecnologias Utilizadas

* **Framework Frontend** : Vue.js 
* **Linguagem** : JavaScript ES6
* **Backend** : Node.js + Express 
* **Estilização** : CSS / Bootstrap / BootstrapVue
* **Gerenciamento de Estado** : Vuex
* **Roteamento** : Vue Router
* **Mock** : Pasta `mock/` para simulação de API durante desenvolvimento
* **Outros** : Axios para consumo de API

## Pré-requisitos

* Node.js / npm ou yarn
